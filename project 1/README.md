
# Python Invoice Program

This is a simple Python program that calculates the total amount for a list of products,
applies a discount if the total is greater than or equal to 100, and prints a detailed invoice.

## Features
- Input product prices
- Calculate total
- Apply discount if total >= 100
- Print detailed invoice

## How to Run
1. Make sure you have Python 3 installed.
2. Run `main.py`.
3. Enter the number of products.
4. Enter the price for each product.
5. See the detailed invoice with total and discount applied.

## Concepts Used
- Input/Output
- Lists
- Loops
- Conditionals (if/else)
- Functions
